가장 많이 사용하는 데이터 타입을 다루는 방법 및 메서드

  • Escape : 백슬래시(\)는 코드로 인식될 수 있는 문자를 문자값으로 변환시키는 역할
  • 문자열 데이터 타입은 시퀀스 특성을 갖고 있으므로 순서를 나타내는 index(0부터 시작)번호를 갖고 있다
  • 문자열 데이터 값 내부에서 개행을 하는 것은 추천하지 않는다
  • 문자열 타입을 객체타입으로 선언 {new String('value');}하는 것은 추후 비교연산자 사용에 어려움을 갖기때문에 추천하지 않는다
  • length프로퍼티는 문자값의 길이를 갖으므로 문자열의 마지막 index 번호를 추정할 때 length-1이라는 공식을 사용하면 된다
